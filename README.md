# Sound Design Web App

## アプリの概要
本アプリケーションは、ブラウザ上で動作するサウンドデザイン・シンセサイザー学習用Webアプリです。
Vue 3, TypeScript, Vite をベースに構築されており、Tone.jsを用いた音声合成やオーディオ録音、波形編集機能などを備えています。
ステップバイステップでオシレーターの操作、エンベロープ（時間変化）、LFO（周期変化）、フィルター、そしてスマートフォン等のジャイロセンサーを利用したインタラクションによる音の操作を学ぶことができます。さらに、マイクからの録音やトリミング、作成した音の保存・共有（URL書き出し、音声ファイル書き出し）も可能です。

## 使い方

### 環境構築・起動方法

Node.js環境が必要です。

1. `app` ディレクトリに移動します。
   ```bash
   cd app
   ```

2. 依存関係をインストールします。
   ```bash
   npm i
   ```

3. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
   (※ `npm run dev` を実行してください)

4. コンソールに表示されるローカルURLにブラウザでアクセスしてください。
※スマートフォン等のジャイロセンサー機能を利用する場合は、同一ネットワーク内からのアクセスなど適宜設定を行ってください。

### ビルド方法

本番環境向けの静的ファイルを生成する場合は以下のコマンドを実行します。

```bash
cd app
npm run build
```
生成されたファイルは `app/dist` フォルダに出力されます。

### アプリの操作手順

アプリ画面の指示に従い、各ステップを進めてください。
- 各パラメータ（オシレーターの波形、ADSR、フィルターのカットオフやレゾナンスなど）を操作して音の変化を確認します。
- 「鳴らす (Hold)」ボタン等を押すことで、リアルタイムに音を確認できます。
- センサー機能やマイク録音をブラウザで許可することで、インタラクティブな操作やサンプルの作成が行えます。
- 最終ステップでは、作成したサウンドを音声ファイルとして録音・ダウンロードしたり、設定をURLとして共有することができます。
